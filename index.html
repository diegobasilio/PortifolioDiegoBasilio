Sobre

Meu nome é Diego de Melo Basilio, sou Desenvolvedor FullStack apaixonado por tecnologias. Em 2022, concluí minha formação em Desenvolvimento de Sistemas pela ETEC Professor Camargo Aranha, onde tive a oportunidade de aprender diversas tecnologias de desenvolvimento, incluindo frontend, backend e banco de dados. Foi durante esse curso que descobri minha afinidade com a área, pois tenho facilidade de aprendizado e grandes habilidades. Atualmente, estou cursando Análise e Desenvolvimento de Sistemas na UNICSUL, buscando aprimorar meu conhecimento na área. Como Desenvolvedor FullStack, tenho experiência tanto na criação de interfaces de usuário atraentes e responsivas usando HTML, CSS e JavaScript, quanto na construção de APIs de tecnologias backend, linguagem de programação Python, Java e conhecimentos em modelagem e gerenciamento de bancos de dados relacionais, como MySQL.
